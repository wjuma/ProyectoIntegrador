Información

Somos estudiantes de la Universidad Politécnica Salesiana que actualmente cursan el quinto semestre la carrera de ingienería de ciencias de la computación por lo cual se ha propuesto desarrollar un proyecto integrador que involucra las materias de:

Software Libre: En la cual se adquiere destresas y conocimientos acerca de lo que es la opensource y todos los criterios que se tomen en cuenta acerca de licencias y este apartado cabe recalcar que software libre no es software gratis. Ademas de haber aprendido a manejar el lenguaje Python.

Sistemas de Informacion: Dentro de este apartado se tuvo una idea mas clara para diferenciar lo que es un sistema informatico y un sistema de informacion. Puesto que un sistema de informacion nos permite solo mostrar informacion respecto a datos obtenidos previamente.

Gestion de Base de Datos: Con lo que respecta a esta asignatura se puede decir que las destrezas dentro de lo que respecta este campo hacen referencia principalmente a diferenciar lo que es una base de datos relacional, una base de datos multidimensional y ademas de tener claro lo que es un markspace y los procedimientos que se han tenido en cuenta para la carga de los datos.
